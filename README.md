# booker-contracts-go

Сгенерированный Go-код из protobuf контрактов.

## ⚠️ Внимание

**Этот репозиторий генерируется автоматически!**

Не редактируй файлы вручную - они будут перезаписаны при следующей генерации.

## Использование

```go
import (
    bookingpb "github.com/bookingcontrol/booker-contracts-go/v1/booking"
    venuepb "github.com/bookingcontrol/booker-contracts-go/v1/venue"
    commonpb "github.com/bookingcontrol/booker-contracts-go/v1/common"
)
```

## Обновление

```bash
go get github.com/bookingcontrol/booker-contracts-go/v1@v1.0.0
go mod tidy
```

## Исходники

Proto файлы находятся в [booker-contracts](https://github.com/bookingcontrol/booker-contracts).
